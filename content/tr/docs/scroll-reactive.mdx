# ScrollReactive

ScrollReactive, görsel hareketini sayfa kaydırma ilerlemesine bağlar. Parallax, giriş animasyonu veya anlatım odaklı bloklar için kullanılır; ilerleme değeri CSS değişkeni olarak sunulur.

## Hızlı Başlangıç

```tsx
import { ReactiveImage } from "reactive-image";

export function ScrollHero() {
  return (
    <ReactiveImage
      variant="scrollReactive"
      src="/hero.jpg"
      alt="Scroll reaktif hero"
      animation="tilt"
      parallaxOffset={80}
      scaleFrom={0.9}
      rotate={10}
      opacityFrom={0.4}
      triggerOffset={120}
      once
    />
  );
}
```

## Animasyonlar

- `fadeIn`: Solma + hafif yukarı kaydırma
- `parallax`: Klasik parallax kaydırma
- `scale`: Kaydırma + ölçekli giriş
- `tilt`: Perspektif tilt + kaydırma/ölçek

## Özellikler

| Prop             | Tip                       | Varsayılan | Açıklama                                                  |
| ---------------- | ------------------------- | ---------- | --------------------------------------------------------- | ------- | ------------ | ----------------- |
| `animation`      | `"fadeIn"                 | "parallax" | "scale"                                                   | "tilt"` | `"parallax"` | Hareket preset'i. |
| `parallaxOffset` | `number`                  | `60`       | Maksimum translate mesafesi (px).                         |
| `scaleFrom`      | `number`                  | `0.92`     | Scale/tilt preset'leri için başlangıç ölçeği.             |
| `rotate`         | `number`                  | `8`        | Tilt preset'i için maksimum rotasyon (derece).            |
| `opacityFrom`    | `number`                  | `0.35`     | Başlangıç opacity değeri.                                 |
| `triggerOffset`  | `number`                  | `0`        | Animasyonun başlaması için offset (px).                   |
| `once`           | `boolean`                 | `false`    | Görsel girildikten sonra ilerlemeyi geri almamaya zorlar. |
| `enableTouch`    | `boolean`                 | `false`    | Dokunma/Pan etkileşimini açar.                            |
| `onEnter`        | `() => void`              | -          | Görsel viewport'a girdiğinde çalışır.                     |
| `onExit`         | `() => void`              | -          | ( `once` değilse ) görsel çıktığında çalışır.             |
| `onProgress`     | `(value: number) => void` | -          | Normalize scroll ilerlemesini (`0-1`) döner.              |

## İpuçları

- Görselin etrafında boşluk bırakmak (padding/margin) animasyonu daha belirgin kılar.
- `className` ile gölge/kenarlık eklerken transform'ların GPU'da kalmasına dikkat edin.
- CSS değişkeni `--ri-scroll-progress` ile kapsama, gradient veya altyazı animasyonlarını tetikleyin.
