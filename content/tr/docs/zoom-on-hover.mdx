# ZoomOnHover

ZoomOnHover varyantı, açık ve çekici ölçekleme efektleri ile görsellerin ayrıntılarını öne çıkarmak için tasarlanmıştır. İmleç takip sistemi, büyüteç etkileri, odak noktası ölçeklendirmesi ve gelişmiş görsel gezinti deneyimleri için pürüzsüz transform animasyonları sunar.

## Özellikler

- **10 Ölçekleme Türü**: Temel yakınlaştırmadan büyüteç efektlere ve lens ölçeklemelere
- **İmleç Takip Sistemi**: Dinamik fare pozisyonu bazlı görsel konumlandırma
- **Transform Origin**: Özelleştirilebilir yakınlaştırma merkez noktaları
- **Büyüteç Efektleri**: Ayrıntılı görsel inceleme için gelişmiş lens görünümleri
- **Dokunmatik Gestures**: Çimdikleme yakınlaştırma ve mobil etkileşim desteği
- **Çoklu Seviye**: Kademeli yakınlaştırma seviyeler ve geçişli deneyimler
- **TypeScript Hazır**: Tam tip tanımları ve IntelliSense desteği

## Hızlı Başlangıç

```tsx
import { ReactiveImage } from "reactive-image";

export default function YakınlaştırılabilirGörsel() {
  return (
    <ReactiveImage
      variant="zoomOnHover"
      src="/ayrıntılı-ürün.jpg"
      alt="Yakınlaştırılabilir ürün görseli"
      animation="smooth"
      scale={2.5}
      transformOrigin="cursor"
    />
  );
}
```

## Olay İşleme & Analitik

Yakınlaştırma etkileşimlerini takip edin ve özel davranışları tetikleyin.

```tsx
export default function TakipEdilen_Yakınlaştırma() {
  const yakınlaştırmaBaşlangıcınıYönet = ({ scale, position }) => {
    console.log("Yakınlaştırma başladı:", { scale, position });
    analytics.track("yakınlaştırma_etkileşimi_başladı", {
      hedef_ölçek: scale,
      başlangıç_pozisyonu: position,
    });
  };

  const yakınlaştırmaDeğişiminiYönet = ({
    currentScale,
    targetScale,
    progress,
  }) => {
    // Gerçek zamanlı yakınlaştırma takibi
    console.log("Yakınlaştırma ilerlemesi:", {
      mevcut: currentScale.toFixed(2),
      hedef: targetScale.toFixed(2),
      ilerleme: `%${(progress * 100).toFixed(1)}`,
    });
  };

  const yakınlaştırmaBitişiniYönet = ({
    finalScale,
    duration,
    interactionType,
  }) => {
    console.log("Yakınlaştırma bitti:", {
      finalScale,
      duration,
      interactionType,
    });
    analytics.track("yakınlaştırma_etkileşimi_bitti", {
      son_ölçek: finalScale,
      etkileşim_süresi: duration,
      etkileşim_türü: interactionType,
    });
  };

  const lensHareketiniYönet = ({ lensPosition, zoomedArea }) => {
    // Büyüteç lens pozisyonu takibi
    console.log(
      "Lens pozisyonu:",
      lensPosition,
      "Yakınlaştırılan alan:",
      zoomedArea
    );
  };

  return (
    <ReactiveImage
      variant="zoomOnHover"
      src="/takip-edilen-yakınlaştırma.jpg"
      alt="Analitik takipli yakınlaştırma"
      animation="magnify"
      scale={3.5}
      transformOrigin="cursor"
      lens={{ enabled: true, size: 160 }}
      onZoomStart={yakınlaştırmaBaşlangıcınıYönet}
      onZoomChange={yakınlaştırmaDeğişiminiYönet}
      onZoomEnd={yakınlaştırmaBitişiniYönet}
      onLensMove={lensHareketiniYönet}
    />
  );
}
```

### Türler

```tsx
// Animasyon türleri
type ZoomAnimationType =
  | "smooth" // Pürüzsüz geçiş yakınlaştırması
  | "instant" // Anında yakınlaştırma
  | "elastic" // Elastik sıçrama efekti
  | "magnify" // Büyüteç lens efekti
  | "focus" // Odak yakınlaştırması
  | "corner" // Köşe yakınlaştırma efekti
  | "multi-level" // Çoklu seviye yakınlaştırma
  | "ken-burns" // Sinematik yavaş yakınlaştırma
  | "wave" // Dalga yakınlaştırma efekti
  | "fragment" // Parça yakınlaştırma
  | "radial" // Radyal yakınlaştırma
  | "pan" // Pan ile yakınlaştırma
  | "layered" // Çoklu katman yakınlaştırması
  | "adaptive"; // Adaptif yakınlaştırma

// Transform origin ayarları
type TransformOrigin =
  | "center" // Merkez noktasından yakınlaştır
  | "cursor" // Fare imleci noktasından yakınlaştır
  | string; // Özel CSS transform-origin değeri (örn: "30% 70%")

// Zamanlama konfigürasyonu
type ZoomTiming = {
  duration?: number; // Milisaniye cinsinden yakınlaştırma süresi (varsayılan: 250)
  reset?: number; // Milisaniye cinsinden sıfırlama süresi (varsayılan: 400)
  delay?: number; // Milisaniye cinsinden başlangıç gecikmesi (varsayılan: 0)
  easing?: string; // CSS yumuşatma işlevi (varsayılan: "ease-out")
};

// Büyüteç lens ayarları
type LensSettings = {
  enabled?: boolean; // Lens efektini etkinleştir (varsayılan: false)
  size?: number; // Lens çapı piksel cinsinden (varsayılan: 100)
  borderWidth?: number; // Lens sınır kalınlığı (varsayılan: 2)
  borderColor?: string; // Lens sınır rengi (varsayılan: "#fff")
  backgroundColor?: string; // Lens arka plan rengi (varsayılan: "transparent")
  cursor?: string; // Lens içi imleç türü (varsayılan: "zoom-in")
};

// Dokunmatik gesture ayarları
type GestureSettings = {
  pinch?: boolean; // Çimdikleme yakınlaştırma (varsayılan: false)
  doubleTap?: boolean; // Çift dokunuş yakınlaştırma (varsayılan: false)
  rotation?: boolean; // Rotasyon gestures (varsayılan: false)
  pan?: boolean; // Pan hareket (varsayılan: false)
  enableTouch?: boolean; // Dokunmatik destek (varsayılan: false)
  maxPinchScale?: number; // Maksimum çimdikleme ölçeği (varsayılan: 4.0)
  minPinchScale?: number; // Minimum ölçek (varsayılan: 1.0)
};

// İmleç takip ayarları
type CursorTrackingSettings = {
  enabled?: boolean; // İmleç takibini etkinleştir (varsayılan: false)
  sensitivity?: number; // Takip hassasiyeti 0-1 (varsayılan: 1.0)
  smoothness?: number; // Hareket yumuşaklığı 0-1 (varsayılan: 0.1)
  boundaries?: {
    x?: [number, number]; // X ekseni sınırları yüzde [-50, 50]
    y?: [number, number]; // Y ekseni sınırları yüzde [-50, 50]
  };
};
```

## Performans Kılavuzları

### Optimizasyon İpuçları

- **Ölçek Aralığı**: Performans için 1.2-3.0, ayrıntılı inceleme için 3.0-5.0 kullanın
- **Büyüteç Lensi**: CPU yoğun, yalnızca gerektiğinde etkinleştirin
- **Mobil**: `scale <= 2.5` kullanın ve karmaşık animasyonlardan kaçının
- **Transform Origin**: `cursor` takip sistem kaynaklarını daha fazla kullanır
- **Gesture Desteği**: Yalnızca gerekli gestures'ları etkinleştirin
