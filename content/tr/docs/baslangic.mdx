## Kurulum

Tercih ettiğiniz paket yöneticisi ile kurun:

```bash
npm install reactive-image
# veya
yarn add reactive-image
# veya
pnpm add reactive-image
```

## Temel Kullanım

Herhangi bir componenti import edip kullanın:

```jsx
import { HoverSwitch } from "reactive-image";

function MyComponent() {
  return (
    <HoverSwitch src="görsel.jpg" hoverSrc="hover-görsel.jpg" alt="Açıklama" />
  );
}
```
