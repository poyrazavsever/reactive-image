## Kurulum

Tercih ettiğiniz paket yöneticisi ile kurun:

```bash
npm install reactive-image
# veya
yarn add reactive-image
# veya
pnpm add reactive-image
```

## Temel Kullanım

Herhangi bir componenti import edip kullanın:

```jsx
import { HoverSwitch } from "reactive-image";

function MyComponent() {
  return (
    <HoverSwitch src="görsel.jpg" hoverSrc="hover-görsel.jpg" alt="Açıklama" />
  );
}
```

## PanReveal Kullanımı

PanReveal iki görseli sinematik kaydırma veya maske ile ortaya çıkarır:

```tsx
import { PanReveal } from "reactive-image";

function SpotlightOrnek() {
  return (
    <PanReveal
      src="/editorial-wide.jpg"
      secondarySrc="/editorial-detail.jpg"
      alt="Detay vurgusu"
      animation="spotlight"
      maskSize={40}
      followCursor
    />
  );
}
```

Önce/sonra senaryoları için `animation="slide"` seçebilir, maskeli açılışlar için `mask` veya `spotlight` modlarına geçip `panAmount`, `maskSize` ve `timing` değerleriyle ince ayar yapabilirsiniz.
