# ClickExpand

ClickExpand varyantı, görselleri sofistike animasyonlar, özelleştirilebilir arkaplanlar ve kapsamlı erişilebilirlik desteği ile etkileşimli modal galerilere dönüştürür. Görsel galeriler, portföyler ve detaylı görsel görüntüleme deneyimleri için mükemmeldir.

## Özellikler

- **9 Animasyon Türü**: Hafif ölçek geçişlerinden dinamik 3D döndürmelere
- **5 Arkaplan Efekti**: Bulanıklık, cam ve özel kaplamalar dahil
- **6 Modal Boyutu**: Kompakt'tan tam görünüm alanı kapsamına
- **Tam Erişilebilirlik**: ARIA uyumluluğu, klavye navigasyonu, odak yakalama
- **Performans Optimizasyonu**: Kaydırma kilitleme, animasyon karesi optimizasyonu
- **TypeScript Desteği**: Tam tip tanımları ve IntelliSense

## Hızlı Başlangıç

```tsx
import { ReactiveImage } from "reactive-image";

export default function Galeri() {
  return (
    <ReactiveImage
      variant="clickExpand"
      src="/galeri-görseli.jpg"
      alt="Güzel manzara"
      modalAnimation="springPop"
      backdrop="blur"
      caption="İsviçre Alplerinde çekildi"
    />
  );
}
```

## Etkileşim Konfigürasyonu

Kullanıcı etkileşimini ve davranışı ince ayarlayın.

```tsx
<ReactiveImage
  variant="clickExpand"
  src="/etkileşimli.jpg"
  alt="Etkileşimli demo"
  // Animasyon ayarları
  modalAnimation="rotateIn"
  animationDuration={350}
  // Etkileşim kontrolleri
  closeOnEsc={true}
  closeOnBackdrop={true}
  enableKeyboard={true}
  preventScroll={true}
  // Olay işleyicileri
  onOpen={() => console.log("Modal açıldı")}
  onClose={() => console.log("Modal kapandı")}
  onAnimationStart={() => console.log("Animasyon başladı")}
  onAnimationEnd={() => console.log("Animasyon tamamlandı")}
/>
```

## Gelişmiş Desenler

### Galeri Izgara

```tsx

  { src: "/galeri3.jpg", alt: "Orman yolu", caption: "Sabah orman yürüyüşü" },

export default function FotoğrafGalerisi() {
  return (
    <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">

        <ReactiveImage

    </div>
  );
}
```

## Özel Hook Kullanımı

Gelişmiş kullanım durumları için alttaki hook'u doğrudan kullanın:

```tsx
import { useClickExpand } from "reactive-image/ClickExpand";

export default function ÖzelModal() {
  const { state, handlers, modalProps } = useClickExpand({
    closeOnEsc: true,
    closeOnBackdrop: true,
    onOpen: () => console.log("Özel modal açıldı"),
    onClose: () => console.log("Özel modal kapandı"),
  });

  return (
    <div>
      <button
        onClick={handlers.open}
        className="bg-blue-500 text-white px-4 py-2 rounded"
      >
        Özel Modal Aç
      </button>

      {state.isOpen && (
        <div
          {...modalProps}
          className="fixed inset-0 bg-black bg-opacity-50 flex items-center justify-center z-50"
          onClick={handlers.handleBackdropClick}
        >
          <div
            className="bg-white p-8 rounded-lg shadow-xl max-w-md w-full mx-4"
            onClick={(e) => e.stopPropagation()}
          >
            <h2 className="text-2xl font-bold mb-4">Özel Modal</h2>
            <p className="text-gray-600 mb-6">
              Maksimum esneklik için ClickExpand hook'u ile oluşturuldu.
            </p>
            <button
              onClick={handlers.close}
              className="bg-gray-500 text-white px-4 py-2 rounded"
            >
              Kapat
            </button>
          </div>
        </div>
      )}
    </div>
  );
}
```

## Erişilebilirlik Özellikleri

ClickExpand erişilebilirlik düşünülerek oluşturulmuştur:

- **ARIA Öznitelikleri**: Uygun `aria-modal`, `role` ve etiketleme
- **Klavye Navigasyonu**: Kapatmak için ESC, açmak için Enter/Space, Tab navigasyonu
- **Odak Yönetimi**: Modal içinde odak yakalama, odak geri yükleme
- **Ekran Okuyucular**: Açıklayıcı etiketler ve duyurular
- **Renk Kontrastı**: Arkaplan kaplamaları yeterli kontrast sağlar

### Klavye Kısayolları

| Tuş                  | Eylem                                |
| -------------------- | ------------------------------------ |
| `Enter` veya `Space` | Görsel odaklanmışken modal'ı aç      |
| `Escape`             | Modal'ı kapat (`closeOnEsc` etkinse) |
| `Tab`                | Modal içinde gezin (odak yakalanmış) |

## API Referansı

### Özellikler

| Özellik             | Tür                  | Varsayılan    | Açıklama                                |
| ------------------- | -------------------- | ------------- | --------------------------------------- |
| `modalAnimation`    | `ModalAnimationType` | `"scaleFade"` | Modal girişi için animasyon türü        |
| `backdrop`          | `BackdropType`       | `"dim"`       | Arkaplan efekt stili                    |
| `caption`           | `string`             | `undefined`   | Görsel altında başlık metni             |
| `closeOnEsc`        | `boolean`            | `true`        | ESC tuşuyla modal'ı kapat               |
| `closeOnBackdrop`   | `boolean`            | `true`        | Arkaplan tıklamasıyla modal'ı kapat     |
| `modalSize`         | `ModalSize`          | `"auto"`      | Maksimum modal boyutları                |
| `animationDuration` | `number`             | `200`         | Milisaniye cinsinden animasyon süresi   |
| `enableKeyboard`    | `boolean`            | `true`        | Klavye navigasyonunu etkinleştir        |
| `preventScroll`     | `boolean`            | `true`        | Modal açıkken body kaydırmasını kilitle |
| `customBackdrop`    | `CSSProperties`      | `undefined`   | Özel arkaplan stilleri                  |
| `modalClassName`    | `string`             | `undefined`   | Modal konteyneri için CSS sınıfı        |
| `captionClassName`  | `string`             | `undefined`   | Başlık için CSS sınıfı                  |
| `onOpen`            | `() => void`         | `undefined`   | Modal açıldığında geri çağırma          |
| `onClose`           | `() => void`         | `undefined`   | Modal kapandığında geri çağırma         |
| `onAnimationStart`  | `() => void`         | `undefined`   | Animasyon başladığında geri çağırma     |
| `onAnimationEnd`    | `() => void`         | `undefined`   | Animasyon bittiğinde geri çağırma       |

### Türler

```tsx
// Animasyon türleri
type ModalAnimationType =
  | "scaleFade"
  | "slideUp"
  | "slideDown"
  | "slideLeft"
  | "slideRight"
  | "springPop"
  | "zoomBounce"
  | "rotateIn"
  | "flipIn";

// Arkaplan türleri
type BackdropType = "dim" | "blur" | "glass" | "dark" | "none";

// Modal boyutları
type ModalSize =
  | "sm" // 400x300px max
  | "md" // 600x450px max
  | "lg" // 800x600px max
  | "xl" // 1200x800px max
  | "full" // %95 görünüm alanı
  | "auto"; // Uyumlu
```

## Performans İpuçları

- Sayfa altındaki görseller için `loading="lazy"` kullanın
- Mobilde büyük modal'ları önlemek için `modalSize`'ı düşünün
- Daha iyi algılanan performans için daha kısa `animationDuration` (150-250ms) kullanın
- Layout kaymaları önlemek için `preventScroll`'u etkinleştirin
- Özel kaplama yönetiminiz varsa `backdrop="none"` kullanın

## Tarayıcı Desteği

- **Modern Tarayıcılar**: Tam destek (Chrome 88+, Firefox 87+, Safari 14+)
- **Odak Yönetimi**: Tüm modern tarayıcılarda desteklenir
- **Backdrop Filter**: Eski tarayıcılarda zarif bozulma
- **CSS Özel Özellikleri**: Yedek değerler sağlanır
