# PolaroidStack

PolaroidStack, birkaç görseli polaroid yığını gibi katmanlar ve hover/dokunuşta hafifçe açar. En üstteki kart erişilebilir kalır, alttakiler ise rastgele rotasyon ve offset ile görsel derinlik sağlar. Kolaj ve portfolyo sunumları için idealdir.

## Hızlı Başlangıç

```tsx
import { ReactiveImage } from "reactive-image";

export function Kolaj() {
  return (
    <ReactiveImage
      variant="polaroidStack"
      src="/cover.jpg"
      alt="Seyahat kolajı"
      stack={[{ src: "/detail-1.jpg" }, { src: "/detail-2.jpg" }]}
      spreadAngle={28}
      offsetStep={16}
      lift={18}
      shadow="strong"
    />
  );
}
```

## Özellikler

| Prop             | Tip                               | Varsayılan             | Açıklama                                |
| ---------------- | --------------------------------- | ---------------------- | --------------------------------------- | --------------------- | ------------------------- |
| `stack`          | `{ src: string; alt?: string }[]` | `[]`                   | Ana görselin altına gelecek ek kartlar. |
| `stackDepth`     | `number`                          | `min(4, stack.length)` | Kaç kart gösterilecek.                  |
| `spreadAngle`    | `number`                          | `32`                   | Yığının toplam açısı.                   |
| `offsetStep`     | `number`                          | `14`                   | Kartlar arası piksel mesafesi.          |
| `lift`           | `number`                          | `14`                   | Aktifken yukarı kaldırma mesafesi.      |
| `rotationJitter` | `number`                          | `4`                    | Kapalıyken hafif rastgele rotasyon.     |
| `shadow`         | `"soft"                           | "medium"               | "strong"`                               | `"medium"`            | Görsel gölge seçenekleri. |
| `aspectRatio`    | `number                           | string`                | `"4 / 3"`                               | Çerçeve en-boy oranı. |
| `enableTouch`    | `boolean`                         | `false`                | Dokunmatik etkileşimi açar.             |
| `onStackEnter`   | `() => void`                      | -                      | Stack aktif olduğunda tetiklenir.       |
| `onStackLeave`   | `() => void`                      | -                      | Stack kapandığında tetiklenir.          |

## İpuçları

- Net bir görünüm için `stackDepth` değerini 3-5 aralığında tutun.
- Görsel oranlarınız farklıysa `aspectRatio` ile hizalayın.
- Marka stilinize uyması için `className` ile kenarlık veya gölge ekleyin.
