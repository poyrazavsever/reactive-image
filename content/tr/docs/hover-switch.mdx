# HoverSwitch

HoverSwitch varyantı, fare üzeri geçişte iki farklı görsel arasında pürüzsüzce geçiş yaparak etkileyici etkileşimler oluşturur. Ürün vitrinleri, önce/sonra karşılaştırmaları ve profesyonel animasyonlar ile performans optimizasyonu içeren etkileşimli galeriler için mükemmeldir.

## Özellikler

- **8 Animasyon Türü**: Pürüzsüz geçişlerden dinamik 3D dönüşlere
- **Performans Optimize**: Görsel ön yükleme, tembel yükleme, verimli yeniden renderlar
- **Dokunmatik Cihaz Desteği**: Mobil deneyimler için isteğe bağlı dokunmatik etkileşimler
- **Özelleştirilebilir Zamanlama**: Süre, gecikme ve yumuşatma üzerinde hassas kontrol
- **Erişilebilirlik Hazır**: Uygun ARIA öznitelikleri ve klavye desteği
- **TypeScript Desteği**: Tam tip tanımları ve IntelliSense

## Hızlı Başlangıç

```tsx
import { ReactiveImage } from "reactive-image";

export default function ÜrünVitrini() {
  return (
    <ReactiveImage
      variant="hoverSwitch"
      src="/ürün-ön.jpg"
      hoverSrc="/ürün-arka.jpg"
      alt="Ürün ön ve arka görünüm"
      animation="slide"
      slideDirection="right"
    />
  );
}
```

## Olay İşleme & Analitik

Etkileşimleri takip edin ve özel davranışları tetikleyin.

```tsx
export default function TakipEdilen_Fare_Üzeri_Görsel() {
  const animasyonBaşlangıcınıYönet = () => {
    // Fare üzeri başlangıcını takip et
    analytics.track("görsel_fare_üzeri_başladı", {
      görsel_türü: "ürün_vitrini",
    });
  };

  const animasyonBitişiniYönet = () => {
    // Fare üzeri tamamlanmasını takip et
    analytics.track("görsel_fare_üzeri_tamamlandı", {
      animasyon_türü: "slide",
    });
  };

  return (
    <ReactiveImage
      variant="hoverSwitch"
      src="/takip-edilen-görsel.jpg"
      hoverSrc="/takip-edilen-fare-üzeri.jpg"
      alt="Analitik takipli görsel"
      animation="slide"
      slideDirection="right"
      onAnimationStart={animasyonBaşlangıcınıYönet}
      onAnimationEnd={animasyonBitişiniYönet}
    />
  );
}
```

## API Referansı

### Özellikler

| Özellik            | Tür                        | Varsayılan    | Açıklama                                                |
| ------------------ | -------------------------- | ------------- | ------------------------------------------------------- |
| `hoverSrc`         | `string`                   | `undefined`   | Fare üzeri durum için görsel kaynağı                    |
| `animation`        | `HoverSwitchAnimationType` | `"crossfade"` | Animasyon türü                                          |
| `slideDirection`   | `SlideDirection`           | `"right"`     | Kaydırma animasyonları için yön                         |
| `timing`           | `HoverSwitchTiming`        | `{}`          | Animasyon zamanlama konfigürasyonu                      |
| `preloadHover`     | `boolean`                  | `true`        | Anında etkileşim için fare üzeri görselini ön yükle     |
| `disableLazy`      | `boolean`                  | `false`       | Kritik görseller için tembel yüklemeyi devre dışı bırak |
| `enableTouch`      | `boolean`                  | `false`       | Dokunmatik cihaz desteğini etkinleştir                  |
| `hoverClassName`   | `string`                   | `undefined`   | Fare üzeri gelindiğinde ek CSS sınıfı                   |
| `onAnimationStart` | `() => void`               | `undefined`   | Animasyon başladığında geri çağırma                     |
| `onAnimationEnd`   | `() => void`               | `undefined`   | Animasyon bittiğinde geri çağırma                       |

### Türler

```tsx
// Animasyon türleri
type HoverSwitchAnimationType =
  | "slide" // Yönlü kaydırma geçişi
  | "crossfade" // Pürüzsüz opaklık çapraz geçişi
  | "zoom" // Geçiş ile ölçek efekti
  | "blur" // Bulanık'tan odağa geçiş
  | "rotateY" // 3D dikey çevirme
  | "slideUp" // Alttan dikey kaydırma
  | "slideDown" // Üstten dikey kaydırma
  | "scaleRotate"; // Birleşik ölçek ve döndürme

// Kaydırma yönleri (kaydırma animasyonu için)
type SlideDirection =
  | "left" // Soldan sağa kaydırma
  | "right" // Sağdan sola kaydırma
  | "up" // Üstten alta kaydırma
  | "down"; // Alttan üste kaydırma

// Zamanlama konfigürasyonu
type HoverSwitchTiming = {
  duration?: number; // Milisaniye cinsinden animasyon süresi (varsayılan: 300)
  delay?: number; // Milisaniye cinsinden animasyon gecikmesi (varsayılan: 0)
  easing?: string; // CSS yumuşatma işlevi (varsayılan: "ease-in-out")
};
```

## Performans Kılavuzları

### Yükleme Stratejisi

- **Kritik Görseller**: `preloadHover={true}` ve `disableLazy={true}` kullanın
- **Sayfa Üstü**: `preloadHover={true}` ve `disableLazy={false}` kullanın
- **Sayfa Altı**: `preloadHover={false}` ve `disableLazy={false}` kullanın
- **Dekoratif**: `preloadHover={false}` ve `disableLazy={false}` kullanın

### Animasyon Performansı

- **Hızlı Etkileşimler**: 200-300ms süre ile `ease-out`
- **Profesyonel Geçişler**: 300-400ms ile `cubic-bezier(0.4, 0, 0.2, 1)`
- **Sanatsal Efektler**: 400-600ms ile `ease-in-out`
- **Mobil Cihazlar**: Daha kısa süreleri tercih edin (200-350ms)

### Bellek Optimizasyonu

- Yalnızca kullanıcıların etkileşim kuracağı fare üzeri görsellerini ön yükleyin
- Uygun görsel formatlarını kullanın (desteklendiğinde WebP, AVIF)
- Görsel boyutlarını göz önünde bulundurun - gereksiz yere büyük görseller yüklemeyin
- Pürüzsüz performans sağlamak için yavaş cihazlarda test edin
