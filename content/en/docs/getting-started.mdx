# Getting Started

Welcome to Reactive Image! This library provides beautiful, interactive React image components with smooth animations.

## Installation

Install using your preferred package manager:

```bash
npm install reactive-image
# or
yarn add reactive-image
# or
pnpm add reactive-image
```

## Basic Usage

Import and use any component:

```jsx
import { ZoomOnHover } from "reactive-image";

function MyComponent() {
  return <ZoomOnHover src="your-image.jpg" alt="Description" zoomScale={1.2} />;
}
```
