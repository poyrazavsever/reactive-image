# TiltOnHover

The TiltOnHover variant creates stunning 3D tilt effects that respond to mouse movement, adding depth and interactivity to images. Features smooth animations, realistic glare effects, gyroscope support for mobile devices, and performance-optimized transforms for premium user experiences.

## Features

- **9 Animation Types**: From basic tilts to magnetic attractions and parallax effects
- **Interactive Glare Effects**: Mouse-following light reflections for premium feel
- **Gyroscope Support**: Mobile device orientation integration
- **Performance Optimized**: RequestAnimationFrame and smooth 60fps rendering
- **Axis Control**: Full control over X, Y, or both-axis movement
- **Touch Support**: Optional touch device interactions
- **TypeScript Ready**: Complete type definitions and IntelliSense

## Quick Start

```tsx
import { ReactiveImage } from "reactive-image";

export default function InteractiveCard() {
  return (
    <ReactiveImage
      variant="tiltOnHover"
      src="/product-card.jpg"
      alt="Interactive product card"
      animation="glare"
      tiltMax={12}
      glare={{
        enabled: true,
        maxOpacity: 0.4,
        position: "mouse",
      }}
    />
  );
}
```

## Event Handling & Analytics

Track tilt interactions and trigger custom behaviors.

```tsx
export default function TrackedTiltCard() {
  const handleTiltStart = ({ tiltX, tiltY }) => {
    console.log("Tilt started:", { tiltX, tiltY });
    // Track interaction start
    analytics.track("tilt_interaction_start");
  };

  const handleTiltMove = ({ tiltX, tiltY, mouseX, mouseY }) => {
    // Real-time tilt tracking
    console.log("Tilt position:", {
      tiltX: tiltX.toFixed(1),
      tiltY: tiltY.toFixed(1),
      mouse: { x: mouseX.toFixed(1), y: mouseY.toFixed(1) },
    });
  };

  const handleTiltEnd = () => {
    console.log("Tilt interaction ended");
    // Track engagement duration
    analytics.track("tilt_interaction_end");
  };

  return (
    <ReactiveImage
      variant="tiltOnHover"
      src="/tracked-tilt.jpg"
      alt="Analytics-tracked tilt"
      animation="glare"
      tiltMax={12}
      glare={{ enabled: true }}
      onTiltStart={handleTiltStart}
      onTiltMove={handleTiltMove}
      onTiltEnd={handleTiltEnd}
    />
  );
}
```

## API Reference

### Props

| Prop            | Type                | Default     | Description                        |
| --------------- | ------------------- | ----------- | ---------------------------------- |
| `tiltMax`       | `number`            | `10`        | Maximum tilt angle in degrees      |
| `animation`     | `TiltAnimationType` | `"basic"`   | Animation type and behavior        |
| `axis`          | `TiltAxis`          | `"both"`    | Tilt axis restriction (both, x, y) |
| `timing`        | `TiltTiming`        | `{}`        | Animation timing configuration     |
| `perspective`   | `number`            | `1000`      | 3D perspective value in pixels     |
| `scale`         | `number`            | `1`         | Scale factor during tilt           |
| `glare`         | `GlareSettings`     | `{}`        | Glare effect configuration         |
| `shadow`        | `boolean`           | `false`     | Enable dynamic shadows             |
| `reverse`       | `boolean`           | `false`     | Reverse tilt direction             |
| `resetOnLeave`  | `boolean`           | `true`      | Reset tilt when mouse leaves       |
| `gyroscope`     | `boolean`           | `false`     | Enable gyroscope support (mobile)  |
| `enableTouch`   | `boolean`           | `false`     | Enable touch device support        |
| `tiltClassName` | `string`            | `undefined` | CSS class applied during tilt      |
| `onTiltStart`   | `function`          | `undefined` | Tilt start callback                |
| `onTiltMove`    | `function`          | `undefined` | Tilt move callback                 |
| `onTiltEnd`     | `function`          | `undefined` | Tilt end callback                  |

### Types

```tsx
// Animation types
type TiltAnimationType =
  | "basic" // Simple 3D tilt effect
  | "glare" // Tilt with interactive light reflection
  | "scale" // Tilt combined with scaling
  | "perspective" // Enhanced 3D depth effect
  | "bounce" // Bouncy reset animation
  | "elastic" // Smooth elastic reset
  | "magnetic" // Attraction-like movement
  | "float" // Floating effect with shadows
  | "parallax"; // Multi-layer movement effect

// Axis control
type TiltAxis =
  | "both" // Full X and Y axis movement
  | "x" // Horizontal movement only
  | "y"; // Vertical movement only

// Timing configuration
type TiltTiming = {
  duration?: number; // Mouse response time in ms (default: 200)
  reset?: number; // Reset animation duration in ms (default: 400)
  easing?: string; // CSS easing function (default: "ease-out")
};

// Glare effect settings
type GlareSettings = {
  enabled?: boolean; // Enable glare effect (default: false)
  maxOpacity?: number; // Maximum glare opacity (default: 0.3)
  color?: string; // Glare color (default: "rgba(255,255,255,0.3)")
  position?: "center" | "mouse"; // Glare positioning (default: "center")
};
```

## Performance Guidelines

### Optimization Tips

- **Tilt Range**: Use 5-15¬∞ for optimal visual impact without motion sickness
- **Perspective**: Keep between 800-1500px for best performance
- **Mobile**: Use `tiltMax <= 10` and avoid complex animations on mobile
- **Glare Effects**: Limit `maxOpacity` to 0.3-0.5 for subtle, professional effects
- **Reset Timing**: 300-600ms provides good balance between responsiveness and smoothness
