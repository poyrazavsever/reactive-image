# HoverSwitch

The HoverSwitch variant creates engaging hover interactions by smoothly transitioning between two different images. Perfect for product showcases, before/after comparisons, and interactive galleries with professional animations and performance optimization.

## Features

- **8 Animation Types**: From smooth crossfades to dynamic 3D rotations
- **Performance Optimized**: Image preloading, lazy loading, efficient re-renders
- **Touch Device Support**: Optional touch interactions for mobile experiences
- **Customizable Timing**: Precise control over duration, delay, and easing
- **Accessibility Ready**: Proper ARIA attributes and keyboard support
- **TypeScript Support**: Full type definitions and IntelliSense

## Quick Start

```tsx
import { ReactiveImage } from "reactive-image";

export default function ProductShowcase() {
  return (
    <ReactiveImage
      variant="hoverSwitch"
      src="/product-front.jpg"
      hoverSrc="/product-back.jpg"
      alt="Product front and back view"
      animation="slide"
      slideDirection="right"
    />
  );
}
```

## Event Handling & Analytics

Track interactions and trigger custom behaviors.

```tsx
export default function TrackedHoverImage() {
  const handleAnimationStart = () => {
    // Track hover start
    analytics.track("image_hover_start", {
      image_type: "product_showcase",
    });
  };

  const handleAnimationEnd = () => {
    // Track hover completion
    analytics.track("image_hover_complete", {
      animation_type: "slide",
    });
  };

  return (
    <ReactiveImage
      variant="hoverSwitch"
      src="/tracked-image.jpg"
      hoverSrc="/tracked-hover.jpg"
      alt="Analytics-tracked image"
      animation="slide"
      slideDirection="right"
      onAnimationStart={handleAnimationStart}
      onAnimationEnd={handleAnimationEnd}
    />
  );
}
```

## API Reference

### Props

| Prop               | Type                       | Default       | Description                                 |
| ------------------ | -------------------------- | ------------- | ------------------------------------------- |
| `hoverSrc`         | `string`                   | `undefined`   | Image source for hover state                |
| `animation`        | `HoverSwitchAnimationType` | `"crossfade"` | Animation type                              |
| `slideDirection`   | `SlideDirection`           | `"right"`     | Direction for slide animations              |
| `timing`           | `HoverSwitchTiming`        | `{}`          | Animation timing configuration              |
| `preloadHover`     | `boolean`                  | `true`        | Preload hover image for instant interaction |
| `disableLazy`      | `boolean`                  | `false`       | Disable lazy loading for critical images    |
| `enableTouch`      | `boolean`                  | `false`       | Enable touch device support                 |
| `hoverClassName`   | `string`                   | `undefined`   | Additional CSS class when hovered           |
| `onAnimationStart` | `() => void`               | `undefined`   | Callback when animation starts              |
| `onAnimationEnd`   | `() => void`               | `undefined`   | Callback when animation ends                |

### Types

```tsx
// Animation types
type HoverSwitchAnimationType =
  | "slide" // Directional slide transition
  | "crossfade" // Smooth opacity crossfade
  | "zoom" // Scale with fade effect
  | "blur" // Blur to focus transition
  | "rotateY" // 3D vertical flip
  | "slideUp" // Vertical slide from bottom
  | "slideDown" // Vertical slide from top
  | "scaleRotate"; // Combined scale and rotate

// Slide directions (for slide animation)
type SlideDirection =
  | "left" // Slide from left to right
  | "right" // Slide from right to left
  | "up" // Slide from top to bottom
  | "down"; // Slide from bottom to top

// Timing configuration
type HoverSwitchTiming = {
  duration?: number; // Animation duration in ms (default: 300)
  delay?: number; // Animation delay in ms (default: 0)
  easing?: string; // CSS easing function (default: "ease-in-out")
};
```

## Performance Guidelines

### Loading Strategy

- **Critical Images**: Use `preloadHover={true}` and `disableLazy={true}`
- **Above the Fold**: Use `preloadHover={true}` and `disableLazy={false}`
- **Below the Fold**: Use `preloadHover={false}` and `disableLazy={false}`
- **Decorative**: Use `preloadHover={false}` and `disableLazy={false}`

### Animation Performance

- **Fast Interactions**: 200-300ms duration with `ease-out`
- **Professional Transitions**: 300-400ms with `cubic-bezier(0.4, 0, 0.2, 1)`
- **Artistic Effects**: 400-600ms with `ease-in-out`
- **Mobile Devices**: Prefer shorter durations (200-350ms)

### Memory Optimization

- Only preload hover images that users are likely to interact with
- Use appropriate image formats (WebP, AVIF when supported)
- Consider image dimensions - don't load unnecessarily large images
- Test on slower devices to ensure smooth performance
