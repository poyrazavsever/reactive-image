# ZoomOnHover

The ZoomOnHover variant creates smooth zoom effects that respond to mouse hover, providing engaging visual feedback and enhanced user interaction. Features multiple animation types, flexible transform origins, customizable effects, and performance-optimized transforms for professional image experiences.

## Features

- **9 Animation Types**: From simple scales to elastic bounces and 3D perspectives
- **Flexible Transform Origins**: 9 predefined positions plus cursor-following zoom
- **Advanced Effects**: Combine zoom with rotation, blur, opacity, and translation
- **Performance Optimized**: Smooth CSS transform animations with 60fps rendering
- **Touch Device Support**: Optional touch interactions for mobile experiences
- **Customizable Timing**: Precise control over duration, delay, and easing
- **TypeScript Ready**: Complete type definitions and IntelliSense

## Quick Start

```tsx
import { ReactiveImage } from "reactive-image";

export default function ProductPreview() {
  return (
    <ReactiveImage
      variant="zoomOnHover"
      src="/product-image.jpg"
      alt="Product preview with zoom"
      animation="elastic"
      zoomScale={1.2}
      origin="center"
      timing={{ duration: 400 }}
    />
  );
}
```

## Event Handling & Analytics

Track zoom interactions and trigger custom behaviors.

```tsx
export default function TrackedZoomImage() {
  const handleZoomStart = () => {
    console.log("Zoom interaction started");
    // Track user engagement
    analytics.track("image_zoom_start", {
      zoom_scale: 1.2,
      animation_type: "elastic",
    });
  };

  const handleZoomEnd = () => {
    console.log("Zoom interaction ended");
    // Track interaction completion
    analytics.track("image_zoom_end");
  };

  return (
    <ReactiveImage
      variant="zoomOnHover"
      src="/tracked-image.jpg"
      alt="Analytics-tracked zoom"
      animation="elastic"
      zoomScale={1.2}
      onZoomStart={handleZoomStart}
      onZoomEnd={handleZoomEnd}
    />
  );
}
```

## API Reference

### Props

| Prop            | Type                | Default     | Description                                 |
| --------------- | ------------------- | ----------- | ------------------------------------------- |
| `zoomScale`     | `number`            | `1.15`      | Zoom scale factor (1.0 = original size)     |
| `animation`     | `ZoomAnimationType` | `"scale"`   | Animation type and behavior                 |
| `origin`        | `ZoomOrigin`        | `"center"`  | Transform origin point                      |
| `timing`        | `ZoomTiming`        | `{}`        | Animation timing configuration              |
| `followCursor`  | `boolean`           | `false`     | Enable cursor following (for cursor origin) |
| `opacityChange` | `number`            | `1`         | Opacity during zoom (0-1)                   |
| `blurAmount`    | `number`            | `0`         | Blur amount in pixels                       |
| `rotation`      | `number`            | `0`         | Rotation angle in degrees                   |
| `containZoom`   | `boolean`           | `false`     | Prevent zoom overflow                       |
| `enableTouch`   | `boolean`           | `false`     | Enable touch device support                 |
| `zoomClassName` | `string`            | `undefined` | CSS class applied during zoom               |
| `onZoomStart`   | `function`          | `undefined` | Zoom start callback                         |
| `onZoomEnd`     | `function`          | `undefined` | Zoom end callback                           |

### Types

```tsx
// Animation types
type ZoomAnimationType =
  | "scale" // Simple scale transformation
  | "scaleRotate" // Scale with rotation effect
  | "scaleBlur" // Scale with blur effect
  | "scaleFade" // Scale with opacity change
  | "scaleSlide" // Scale with slight translation
  | "perspective" // 3D perspective transformation
  | "pulse" // Quick pulse effect
  | "bounce" // Bouncy elastic effect
  | "elastic"; // Smooth elastic transformation

// Transform origins
type ZoomOrigin =
  | "center" // Center of image
  | "top" // Top edge
  | "bottom" // Bottom edge
  | "left" // Left edge
  | "right" // Right edge
  | "top-left" // Top-left corner
  | "top-right" // Top-right corner
  | "bottom-left" // Bottom-left corner
  | "bottom-right" // Bottom-right corner
  | "cursor"; // Mouse position

// Timing configuration
type ZoomTiming = {
  duration?: number; // Animation duration in ms (default: 300)
  delay?: number; // Animation delay in ms (default: 0)
  easing?: string; // CSS easing function (default: "ease-out")
};
```

## Performance Guidelines

### Optimization Tips

- **Scale Range**: Use 1.1-1.3 for optimal visual impact without performance issues
- **Mobile Optimization**: Prefer shorter durations (200-400ms) and smaller scales on mobile
- **Blur Effects**: Limit blur to 2-5px for good performance across devices
- **Container Overflow**: Use `containZoom={true}` to prevent layout shifts
- **Touch Events**: Enable `enableTouch` only when necessary to save resources
