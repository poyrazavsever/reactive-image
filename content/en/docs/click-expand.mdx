# ClickExpand

The ClickExpand variant transforms images into interactive modal galleries with sophisticated animations, customizable backdrops, and comprehensive accessibility support. Perfect for image galleries, portfolios, and detailed image viewing experiences.

## Features

- **9 Animation Types**: From subtle scale fades to dynamic 3D flips
- **5 Backdrop Effects**: Including blur, glass, and custom overlays  
- **6 Modal Sizes**: From compact to full viewport coverage
- **Full Accessibility**: ARIA compliance, keyboard navigation, focus trapping
- **Performance Optimized**: Scroll locking, animation frame optimization
- **TypeScript Support**: Complete type definitions and IntelliSense

## Quick Start

```tsx
import { ReactiveImage } from "reactive-image";

export default function Gallery() {
  return (
    <ReactiveImage
      variant="clickExpand"
      src="/gallery-image.jpg"
      alt="Beautiful landscape"
      modalAnimation="springPop"
      backdrop="blur"
      caption="Captured in the Swiss Alps"
    />
  );
}
```

## Custom Hook Usage

For advanced use cases, use the underlying hook directly:

```tsx
import { useClickExpand } from "reactive-image/ClickExpand";

export default function CustomModal() {
  const { state, handlers, modalProps } = useClickExpand({
    closeOnEsc: true,
    closeOnBackdrop: true,
    onOpen: () => console.log('Custom modal opened'),
    onClose: () => console.log('Custom modal closed'),
  });

  return (
    <div>
      <button 
        onClick={handlers.open}
        className="bg-blue-500 text-white px-4 py-2 rounded"
      >
        Open Custom Modal
      </button>

      {state.isOpen && (
        <div 
          {...modalProps}
          className="fixed inset-0 bg-black bg-opacity-50 flex items-center justify-center z-50"
          onClick={handlers.handleBackdropClick}
        >
          <div 
            className="bg-white p-8 rounded-lg shadow-xl max-w-md w-full mx-4"
            onClick={(e) => e.stopPropagation()}
          >
            <h2 className="text-2xl font-bold mb-4">Custom Modal</h2>
            <p className="text-gray-600 mb-6">
              Built with the ClickExpand hook for maximum flexibility.
            </p>
            <button 
              onClick={handlers.close}
              className="bg-gray-500 text-white px-4 py-2 rounded"
            >
              Close
            </button>
          </div>
        </div>
      )}
    </div>
  );
}
```

## Accessibility Features

ClickExpand is built with accessibility in mind:

- **ARIA Attributes**: Proper `aria-modal`, `role`, and labeling
- **Keyboard Navigation**: ESC to close, Enter/Space to open, Tab navigation
- **Focus Management**: Focus trapping within modal, focus restoration
- **Screen Readers**: Descriptive labels and announcements
- **Color Contrast**: Backdrop overlays ensure sufficient contrast

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Enter` or `Space` | Open modal when image focused |
| `Escape` | Close modal (if `closeOnEsc` enabled) |
| `Tab` | Navigate within modal (focus trapped) |

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `modalAnimation` | `ModalAnimationType` | `"scaleFade"` | Animation type for modal entrance |
| `backdrop` | `BackdropType` | `"dim"` | Backdrop effect style |
| `caption` | `string` | `undefined` | Caption text below image |
| `closeOnEsc` | `boolean` | `true` | Close modal on ESC key |
| `closeOnBackdrop` | `boolean` | `true` | Close modal on backdrop click |
| `modalSize` | `ModalSize` | `"auto"` | Maximum modal dimensions |
| `animationDuration` | `number` | `200` | Animation duration in milliseconds |
| `enableKeyboard` | `boolean` | `true` | Enable keyboard navigation |
| `preventScroll` | `boolean` | `true` | Lock body scroll when modal open |
| `customBackdrop` | `CSSProperties` | `undefined` | Custom backdrop styles |
| `modalClassName` | `string` | `undefined` | CSS class for modal container |
| `captionClassName` | `string` | `undefined` | CSS class for caption |
| `onOpen` | `() => void` | `undefined` | Callback when modal opens |
| `onClose` | `() => void` | `undefined` | Callback when modal closes |
| `onAnimationStart` | `() => void` | `undefined` | Callback when animation starts |
| `onAnimationEnd` | `() => void` | `undefined` | Callback when animation ends |

## Performance Tips

- Use `loading="lazy"` for images below the fold
- Consider `modalSize` to prevent oversized modals on mobile
- Use shorter `animationDuration` (150-250ms) for better perceived performance
- Enable `preventScroll` to prevent layout shifts
- Use `backdrop="none"` if you have custom overlay handling

## Browser Support

- **Modern Browsers**: Full support (Chrome 88+, Firefox 87+, Safari 14+)
- **Focus Management**: Supported in all modern browsers
- **Backdrop Filter**: Graceful degradation in older browsers
- **CSS Custom Properties**: Fallback values provided
